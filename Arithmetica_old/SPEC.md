## 1. タイトル・世界観・キャラクター

- **タイトル**：アリスメティカ（Arithmetica）
- **世界観**：「アリスメティカ」という古代の数字文明が残した遺跡・塔が舞台。数字の力を使いこなす者だけが最深部に辿り着けるとされている。プレイヤーは数字カードを操る冒険者として塔に挑む。
- **ストーリー**：シンプルに留め、ゲームプレイを中心に据える。
- **主人公**：男の子・女の子の2種類からゲーム開始時に選択。ゲームプレイへの影響なし（見た目のみ）。ビジュアルは後日差し替え予定。

## 2. 基本ステータス

| ステータス | 内容 |
|-----------|------|
| HP | 初期20、最大HP拡張で上昇 |
| 鎧 | 永続防御値。無期限バフで強化 |
| 闘志 | 攻撃式の結果にプラスされる永続値 |
| お金 | 敵討伐で入手、商店等で使用 |

## 3. ダメージ計算（3層構造）

敵の攻撃が来たとき、以下の順番で処理する。

1. **防御力**（引き算の式の結果）で吸収
2. 残りを**鎧**で吸収
3. さらに残ったダメージが**HPに直撃**

**例）** 敵攻撃15、防御力8、鎧3の場合：
`15 - 8（防御力）= 7 → 7 - 3（鎧）= 4 → HP - 4`

---

## 4. 攻撃計算の処理順

1. 足し算の式の計算結果を出す
2. 加算効果を全て足す
3. 闘志を加算
4. 倍率を適用

---

## 5. 戦闘基本ルール

| 項目 | 内容 |
|------|------|
| 手札 | 常に5枚キープ（使った分だけ補充） |
| 式 | 毎ターン2つ作る。演算子は各式で自由に選択 |
| 足し算の式 | 攻撃ダメージになる |
| 引き算の式 | 防御力になる（毎ターンリセット） |
| デッキ | 1〜9を2枚ずつ、計18枚 |
| 捨て札処理 | 山札が尽きたら捨て札をシャッフルして循環 |
| 封印 | デバフによる封印は最大1枚まで |

### 戦闘の流れ（1ターン）

1. 手札5枚が表示される（封印カードはグレーアウト）
2. 式①を作る（使うカードを選び、演算子を選ぶ）
3. 式②を作る（同上）
4. GOボタンを押す
5. 左から順番に特殊能力が発動
6. 攻撃・防御の結果を適用
7. 敵の行動が発動
8. 使ったカードは捨て札へ、手札を5枚に補充

## 6. カードへの能力付与ルール

| 項目 | 内容 |
|------|------|
| スロット数 | 基本2スロット。神殿でお金を払うと最大3まで拡張可能 |
| 付与数 | 1枚のカードに最大2つまで能力を付与可能 |
| 発動順 | 左から順番に処理 |
| 倍率の適用 | 加算効果・闘志の後に適用 |
| 適用範囲 | 効果はその式のみ（2式またぎ効果を除く） |

## 7. 数字の性質と色分け（1〜9）

| 色 | 性質 | 対象数字 |
|----|------|---------|
| 🔴 赤 | 偶数 | 2, 4, 6, 8 |
| 🔵 青 | 奇数 | 1, 3, 5, 7, 9 |
| 🟡 黄 | 素数 | 2, 3, 5, 7 |
| 🟢 緑 | 合成数 | 4, 6, 8, 9 |

## 8. 特殊能力

### 特殊能力の処理順（全能力共通）
1. 式の計算結果を出す
2. 加算効果を全て足す（2式またぎ効果も含む）
3. 闘志を加算（攻撃式のみ）
4. 倍率を適用

### 8-1. バフ系（持ち込み選択）

| 名前 | 効果 | 持続 | 適用 |
|------|------|------|------|
| 強化 | 攻撃または防御＋3 | このターン | その式のみ |
| 回復 | HP＋2 | 即時 | － |
| 連続回復 | HP＋1 | 3ターン | － |
| 鎧回復 | 鎧＋2 | 即時 | － |
| 鎧連続回復 | 鎧＋1 | 3ターン | － |
| 盾持ち越し（短） | 残った防御力を全て次ターンに持ち越す | 1ターン | － |
| 盾持ち越し（長） | 残った防御力の半分を持ち越す | 3ターン | － |
| 攻撃2倍(1/2) | 同じ式で2回発動で攻撃2倍 | このターン | 攻撃の式のみ |
| 攻撃3倍(1/3) | 同じ式で3回発動で攻撃3倍 | このターン | 攻撃の式のみ |
| 防御2倍(1/2) | 同じ式で2回発動で防御2倍 | このターン | 防御の式のみ |
| 防御3倍(1/3) | 同じ式で3回発動で防御3倍 | このターン | 防御の式のみ |

### 8-2. シナジー系（戦闘中に獲得）

#### 計算結果トリガー（攻撃・防御どちらの式でも発動）

| 名前 | 発動条件 | 効果 |
|------|---------|------|
| 偶数の力 | 答えが偶数 | ＋6 |
| 奇数の力 | 答えが奇数 | ＋6 |
| 素数の力 | 答えが素数 | ＋8 |
| 3の刻印 | 答えが3の倍数 | ＋3、HP＋2 |
| 5の加護 | 答えが5の倍数 | ＋5、HP＋3 |
| 7の鼓動 | 答えが7の倍数 | ＋7、HP＋4 |

#### 使用カードトリガー（攻撃・防御どちらの式でも発動）

| 名前 | 発動条件 | 効果 |
|------|---------|------|
| 偶数コンボ | 式のカードが全て偶数 | ＋7 |
| 奇数コンボ | 式のカードが全て奇数 | ＋7 |
| 素数コンボ | 式のカードが全て素数 | ＋5、HP＋3 |
| 連番コンボ | 式のカードが連続した数字 | ＋8 |
| 同数コンボ | 式に同じ数字が2枚 | ＋6 |
